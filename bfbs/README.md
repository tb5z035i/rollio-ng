# BFBS Schemas

Prebuilt binary FlatBuffer reflection schemas (`.bfbs`) used by
`rollio-episode-mcap` to register channel schemas in MCAP files.

These files are committed to the repo. To regenerate from the `.fbs` sources
in `utils/schemas/`, run `utils/regenerate-bfbs.sh` (requires `flatc` 25.12.19).
